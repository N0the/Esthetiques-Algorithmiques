# Fractale

## Projet

Pour ce projet, j'ai décidé de faire un fractale en 3D. Pour ce faire, j'ai tout d'abord suivi un tutoriel sur youtube me permettant de réaliser ce projet (lien présent dans les crédits). J'ai par la suite eu l'idée de modifier un peu la génération de ce fractale (notamment sur la forme de celui-ci) et de le répéter afin de former un motif en 3D (ici un oeil), tout en y ajoutant de la couleur !

Il est à préciser que la navigation se fait à la souris, avec le clique gauche/droit pour se déplacer, la molette pour la profondeur et le clique molette pour bouger la scène.

Vous pouvez si vous voulez, manipuler les variables d'initialisation comme dim, number ou translateNumber (attention à ne pas mettre de valeurs trop élevées)

## Exécution

A exécuter sur processing, il faut également posséder la library PeasyCam, pour ce faire aller dans l'onglet outil, puis manage tools, chercher dans l'onglet library "PeasyCam" de Jonathan Feinberg et installer la library.

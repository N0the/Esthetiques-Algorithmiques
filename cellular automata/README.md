# Automate cellulaire

## Projet

Pour ce projet, j'ai décidé de faire un automate cellulaire en trois dimensions. Pour ce faire, j'ai tout d'abord suivi un tutoriel sur youtube me permettant de réaliser ce projet (lien présent dans les crédits).
Il y a également des boutons d'actions : 
- Les boutons du numpad, de 1 à 9 qui permettent d'augmenter la taille des cellules générées
- Le + et le - du numpad qui permettent d'accélérer ou de ralentir respectivement la génération
- La barre espace pour mettre en pause le code

Il est à préciser que la navigation se fait à la souris, avec le clique gauche/droit pour se déplacer, la molette pour la profondeur et le clique molette pour bouger la scène.

## Exécution

A exécuter sur processing, il faut également posséder la library PeasyCam, pour ce faire aller dans l'onglet outil, puis manage tools, chercher dans l'onglet library "PeasyCam" de Jonathan Feinberg et installer la library.

# Littérature numérique

## Projet

Ce projet est un générateur de poèmes à partir des choix de l'utilisateur, entre les différents vers cliquable, qui seront à la fin, concaténer puis afficher à l'écran.

## Exécution

Importer le fichier twee sur twine : https://twinery.org/
Et lancer le bouton play dans build

Ou alors ouvrir le fichier html
